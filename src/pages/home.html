Delivery Report Extractor

Extracts structured information from AI development delivery reports.

What this app does

This app extracts structured information from delivery report text.

It identifies:

  • summary
  • changed files
  • implemented items
  • fixed items
  • tests
  • notes

When ChatGPT should use it

ChatGPT should use this app when a user provides an AI development delivery report and wants consistent structured information extracted.

What input it accepts

Input:

Delivery report text.

Optional:

source label.

What output it returns

Returns structured output:

  • status
  • summary
  • changed_files
  • implemented_items
  • fixed_items
  • tests
  • notes
  • errors

Available tools

Available tools:

extract_delivery_report

MCP endpoint

MCP endpoint:

POST /mcp

What this app does NOT do

This app does NOT:

  • modify code
  • deploy projects
  • connect to GitHub
  • execute development tasks
  • approve changes

Data handling

Submitted delivery report text is processed only for extraction.

No persistent storage is used.
