Privacy

Data collected

The app processes delivery report text submitted by users.

How input is used

Input is used only to extract structured delivery information.

How output is generated

Output is generated through predefined extraction logic that organizes report sections into structured fields.

Retention

The app is stateless.

No delivery reports or extracted results are permanently stored.

External sharing

No submitted content is shared with external services.

External API policy

This app does not call external APIs.

Account / login policy

No account creation or login is required.

User controls

Users control the information they submit and may choose what content to provide.

Read-only boundary

The app only extracts information.

It does not modify external systems.
